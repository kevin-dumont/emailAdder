Démo EmailAdder Plugin jQuery

Créez facilement vos formulaires d'envoi de mail.

GitHub

Utilisation classique

Entrez un e-mail puis insérez un espace, une virgule, un tiret, ou un underscore. Recommencez, supprimez-en un, puis deux. Copiez-collez en un, pas besoin d'espace.

E-mails par défaut

Utilisez l'attribut data-default pour spécifier une/plusieurs adresse(s) mail visibles dés le chargement de la page. Si les adresses ne sont pas du format e-mail, elles ne seront pas affichées.

Obtenir les données pour les traitements ajax

Les données issues des champs emailAdder se récupèrent en javascript via la fonction jQuery emailAdderData() qui retournera un tableau contenant les emails entrés.

$(document).ready(function(){
	$("Votre sélecteur").emailAdderData();
});

Note : Pour la récupération en php, rien n'est à faire, un tableau sera disponible dans $_POST, ayant pour index l'attribut name du champ.
